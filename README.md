# ShadowBridge

A silent mobile control bridge running in the shadows.

* Install termux & termux:api app.
* Give all permission on termux:api app.
* After open termux app & run this commands one by one.

$ termux-setup-storage
$ pkg update
$ pkg upgrade -y
$ pkg install git -y
$ pkg install python -y
$ pkg install termux-api
$ pkg install cloudflared -y
$ pip install colorama
$ git clone https://github.com/his-bullah/ShadowBridge.git
$ unzip ShadowBridge.zip
$ cd ShadowBridge
$ python Shadow.py settings
